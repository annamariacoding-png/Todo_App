# Todo_App

A terminal-based Todo application built with **Python** and **MySQL** that helps users manage their daily tasks efficiently. Users can add, complete, remove, and view tasks, while maintaining a history of completed tasks.

 **Features**

- Add new tasks
- View all pending tasks
- Remove unwanted tasks
- View completed task history
- Stores data using a MySQL database

 **Technologies Used**

- Python
- MySQL
- mysql-connector-python
- Git
- GitHub
